Hey! I'm Sneha. Welcome to my blog.

The title of this blog is based on [a piece](/blog/2022/01/hurtling_through_space/) I wrote almost 3 years ago today (I'm writing this about page in December 2024). I often go back to it during difficult times, remind myself of how fragile everything around me is, and maybe take some time to rethink how to reach my mission control.

This is all a work in progress, as we all always are I'm sure. But this is my attempt at putting my thoughts, ramblings, and musings out there. I've got a lot of ideas, some of them coherent ramblings, some of them absolutely absurd lines of thought. Whatever the thoughts are, they are all me. No AI is used in the writing of this blog.

## Technical Stack and Housekeeping

The source code for this site is available on [GitHub](https://github.com/snehasaisneha/saisneha.com). It's currently built using [Astro](https://astro.build/) and deployed on Netlify. Older versions of this site used Hugo, and older still, Jekyll. The code is not licensed because I haven't yet worked out how I want to separate code licensing from the text itself.
