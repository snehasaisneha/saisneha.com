Hey! I'm Sneha. Welcome to my blog.

The title of this blog is based on [a piece](/2022/01/hurtling_through_space/) I wrote almost 3 years ago today (I'm writing this about page in December 2024). I often go back to it during difficult times - remind myself of how fragile everything around me is, and maybe take some to rethink how to reach my mission control.

This is all a work in progress, as we all always are I'm sure. But this is my attempt at putting my thoughts, ramblings, and musings out there. I've got a lot of ideas - some of them coherent ramblings, some of them absolutely absurd lines of thought. Whatver the thoughts are, they are all me (no AI is used in the writing of this blog).

## Subscribe
You can subscribe to my ramblings below using your email address, and the power users among you can also subscribe using the RSS feed at [saisneha.com/feed](/feed). The mailing list is hosted on ConvertKit, and is subject to their [privacy policy](https://convertkit.com/privacy).

## Technical Stack and Housekeeping
The source code for this site is available at [github](https://github.com/snehasaisneha/saisneha.com). It's built using hugo, and a template from [carsonjp](https://github.com/carsonip/hugo-theme-minos). Older versions of this site were using jekyll, but that's just gotten unreasonably slow. The deploy is done using netlify. The code is not licensed because I honestly don't know how to freely license the underlying code without licensing the text itself. Maybe I'll build a fork of the base template with my modifications and release it under GPLv3 or MIT.
